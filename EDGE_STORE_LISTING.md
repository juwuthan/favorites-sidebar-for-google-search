# Microsoft Edge Add-ons Listing

## Title

Favorites Sidebar for Google Search

## Published Listing

https://microsoftedge.microsoft.com/addons/detail/favorites-sidebar-for-goo/fmagimipbmjoafpegccdgabmggoolfnp

## Short Description

Shows your Edge favorites beside Google Search and opens new tabs on Google.com.

## Full Description

Favorites Sidebar for Google Search gives Google Search a focused, browser-native favorites area. On wide Google Search home pages, it displays the links from your Edge favorites bar in a compact left sidebar and shifts the search page to the right. On search results pages and smaller windows, it switches to a small launcher button so the search interface stays usable.

The extension also redirects new tabs to https://www.google.com/, making Google Search the default new-tab destination.

This extension is an independent project and is not affiliated with, endorsed by, sponsored by, or approved by Google LLC, Microsoft Corporation, or the Microsoft Edge team.

Key features:

- Display Edge favorites beside Google Search.
- Keep the sidebar open on wide Google Search home pages.
- Use a compact launcher on search results pages and small windows.
- Open new tabs directly on Google.com.
- Use local browser favicons where available.
- Replace missing favicons with simple pastel dots.
- Run only on Google Search pages, not on Gmail, Calendar, Google Photos, Drive, or other Google apps.

Permissions are limited to what the extension needs: bookmarks access to read the favorites bar, favicon access to show local browser favicons, Google Search host permissions for the UI, and the standard new-tab override to open Google.com.

The new-tab change is reversible at any time by disabling or removing the extension from edge://extensions. The extension does not include analytics, ads, tracking code, or a remote backend.

Bookmark titles and URLs are rendered into the Google Search page while the sidebar or launcher is visible. The extension does not send bookmark data to an extension developer server.

## Certification Notes

The extension is intended to run only on Google Search pages:

- `https://www.google.com/`
- `https://www.google.com/search...`
- `https://www.google.de/`
- `https://www.google.de/search...`

New tabs are redirected to `https://www.google.com/`.

The extension uses the standard `chrome_url_overrides.newtab` API. Users can reverse the new-tab change by disabling or removing the extension from `edge://extensions`.

The extension reads the browser favorites bar locally and renders favorite titles/URLs into Google Search pages only. It does not run on Gmail, Google Calendar, Google Photos, Google Drive, or other Google apps.

The extension does not use remote code, analytics, ads, or a developer-operated backend.
